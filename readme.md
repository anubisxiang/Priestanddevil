#项目说明
##1，各代功能
boats脚本实现船只的设计，用向量来保存角色
coasts实现河岸，用向量实现对角色的操作
UserGUI在游戏中进行游戏说明，并在游戏结束时打印出说明文字
Director创建一个instance的实例
FirstController实现加载游戏场景资源，游戏中船只移动，游戏结束的判断
Interfaces实现相关接口
##2
运行方式
首先下载资源，然后创建一个空对象，再把FirstController拖拽到对象的component即可点击运行游戏
